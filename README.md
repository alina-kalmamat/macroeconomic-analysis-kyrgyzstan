# Macroeconomic Analysis of Kyrgyzstan

## Project Overview

This project analyzes macroeconomic indicators in Kyrgyzstan between 2010 and 2024 using publicly available economic data.

The analysis focuses on:
- USD/KGS exchange rate
- Inflation index
- Average monthly salary

The project includes data cleaning, aggregation, exploratory data analysis (EDA), correlation analysis, and data visualization.

## Objectives

The main goals of this project are:
- Analyze long-term trends in key macroeconomic indicators
- Explore relationships between exchange rates, inflation, and salaries
- Practice real-world data cleaning and analysis workflows using Python

## Dataset Sources

Data was collected from publicly available sources:
- National Bank of the Kyrgyz Republic
- National Statistical Committee of the Kyrgyz Republic

Datasets:
- Exchange rates (2010–2024)
- Inflation index
- Average monthly salary
